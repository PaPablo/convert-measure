# convert-measure

A Clojure library designed to convert measures given a certain format of CLI
parameters

## Usage

To use `convert-measure` run with lein:

```bash
$ lein run 1 g to oz
```

## Conversions currently supported

* kg -> g 
* g -> kg
* oz -> g 
* g -> oz
* c -> f
* f -> c

More to come...
